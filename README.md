# SoundFlow Studio

A full-stack SaaS dashboard for audio management, playlist creation, and audio insights.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```

The app will be available at `http://localhost:3000`

## Features

- Audio file upload and management
- Playlist creation and management
- Audio insights and analysis
- User authentication
- Responsive design

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- React Query
- Supabase (optional)
